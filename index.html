Projects

  • LearnBot: A Streamlit app that creates instant courses, interactive quizzes, and features an AI assistant for chatting with videos and documents.
  • Pytorch Neural Network Visualizer: A tool designed to provide a dynamic visualization of neural network training. It helps beginners understand what is happening under the hood of the training process.
  • QuantaGrad: An Autograd engine built for fun. Implements backpropagation and a small neural networks library on top of it with a PyTorch-like API.
  • Robotics Project: Simulated PID control of a drone in MATLAB, implemented motion planning algorithms such as A* and Dijkstra for path planning, utilized Gaussian Mixture Models for color detection, applied Kalman filters to reduce noise in LiDAR data, and simulated occupancy grid mapping using MATLAB.
  • Classical ML Algorithms: Implemented classical machine learning algorithms, including linear regression, logistic regression, TF-IDF vectorization, Naive Bayes classifier, and techniques for sentiment analysis and sentence similarity.
